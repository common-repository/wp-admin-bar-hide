WordPress Adminbar hide from frontend side.

== Description ==

This Plugin Allow to hide WP Adminbar in website frontend side.




== Installation ==

This section describes how to install the plugin and get it working.


1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'WP Adminbar Hide' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin
4. Enjoy!, there are no settings for this.
